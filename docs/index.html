Privacy Policy for AW_MediaQS

Last updated: 2025

Data Collection

The extension does NOT collect or transmit:

  • Personal user data
  • Browser history
  • User files
  • Information from visited websites
  • Any tracking information

Data Storage

All settings are stored locally in your browser using Chrome Storage API.

Permissions

  • activeTab - access to current tab for media detection
  • storage - save extension settings
  • downloads - download media files
  • clipboardWrite - copy links to clipboard
  • clipboardRead - copy media files to clipboard

Contact

For questions: https://github.com/2biteWolf/AW_mediaQS/issues

Политика конфиденциальности AW_MediaQS

Последнее обновление: 2024

Сбор данных

Расширение НЕ собирает и не передает:

  • Личные данные пользователя
  • Историю браузера
  • Файлы пользователя
  • Информацию с посещаемых сайтов
  • Данные для отслеживания

Хранение данных

Все настройки сохраняются локально в браузере пользователя через Chrome Storage API.

Разрешения

  • activeTab - доступ к текущей вкладке для обнаружения медиа
  • storage - сохранение настроек расширения
  • downloads - загрузка медиафайлов
  • clipboardWrite - копирование ссылок в буфер
  • clipboardRead - копирование медиафайлов в буфер
